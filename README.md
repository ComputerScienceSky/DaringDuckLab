# DaringDuckLab
A variation of the Busy Beaver Problems with a Turing Machine Runner that I created. 

Score = Number of 1s / (Number of States + Alphabet + Input Cardinality)

Score = 1111001 / (103 States + 3 Alphabet + 11000 input) = 100!
